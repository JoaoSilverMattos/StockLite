# StockLite: Análise e Modelagem de Sistema de Gerenciamento de Estoque

## Visão Geral do Projeto

Este repositório contém a análise e modelagem do sistema StockLite, um sistema de gerenciamento de estoque desenvolvido como parte do projeto de Práticas Extensionistas III do curso de Análise e Desenvolvimento de Sistemas. O objetivo principal é demonstrar a aplicação de conceitos de modelagem UML (Diagramas de Entidade e Relacionamento, Casos de Uso, Classes, Atividades e Sequência) para a concepção de um sistema robusto e funcional.

## Conteúdo do Repositório

O repositório inclui os seguintes artefatos de modelagem:

*   **Diagrama de Entidade e Relacionamento (DER):** Detalha a estrutura do banco de dados, focando nas entidades `USUARIO` e `PRODUTO` e seus relacionamentos.
*   **Diagrama de Caso de Uso Geral:** Apresenta as funcionalidades do sistema sob a perspectiva do ator principal, `fabio_jean`, e os casos de uso como `Efetuar Login`, `Cadastrar`, `Editar/Excluir Item`, `Gerar Relatórios` e `Visualizar Itens`.
*   **Diagrama de Classes:** Estrutura estática do sistema, mostrando as classes `Produto` e `Usuario`, seus atributos, métodos e relacionamentos.
*   **Diagrama de Atividades:** Ilustra o fluxo de trabalho para o processo de cadastro de um item no sistema StockLite.
*   **Diagrama de Sequência (Fluxo de Login):** Detalha a interação entre os objetos do sistema durante o processo de login.
